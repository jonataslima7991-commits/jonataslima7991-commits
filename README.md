## 📁 Experiência Acadêmica & Projetos

### 🔬 Revisão Sistemática: IA e Meio Ambiente — 1ª Mostra Acadêmica FATEC | Setembro 2025

- **Objetivo:** Mapear aplicações de IA no setor ambiental (2013–2025) para identificar tendências e oportunidades sustentáveis
- **Metodologia:** Framework CRISP-DM para estruturação e limpeza dos dados
- **Ferramentas:** Publish or Perish · VOSviewer · Microsoft Excel
- **Resultado:** Identificação de clusters de tecnologia verde e produção de relatório técnico sobre o avanço da pesquisa científica no setor

### 🔍 A Engenharia Reversa da Busca de Emprego: Uma análise sobre o Mercado de Dados Brasileiro — 2ª Mostra Acadêmica FATEC | Junho 2026

- **Objetivo:** Mapear quantitativamente o mercado de trabalho em Ciência de Dados no Brasil, aplicando engenharia reversa para revelar a real distribuição de cargos, nomenclaturas mascaradas e remunerações — desconstruindo a opacidade estrutural do ecossistema de vagas.

- **Metodologia:** Coleta automatizada via web scraping com Python e Selenium na plataforma Adzuna.com.br (agregador de LinkedIn, Gupy, Catho e Indeed). Processamento e limpeza dos dados com Pandas seguindo o framework CRISP-DM. Visualização final em dashboard HTML e Power BI.

- **Resultado:** Buscar por "Dados" abre 6,3x mais portas que "Cientista de Dados". 93,2% das vagas omitem salário. Cargos com remuneração exposta variam de R$ 3.561 (Analista) a R$ 9.150 (Especialista). O estudo aponta a ausência de padronização na publicação de vagas como barreira real ao acesso a oportunidades no mercado de tecnologia brasileiro.
